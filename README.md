# 🛡️ Cyber Risk Register

**By Priyanshu | Cybersecurity & GRC Specialist**

An interactive, browser-based Cyber Risk Register tool built to demonstrate practical GRC knowledge. No installation or server required — open the HTML file and start logging risks immediately.

---

## 🚀 Live Demo

> Download `cyber-risk-register.html` → open in any browser → it works instantly.

---

## ✨ Features

- **Add & track risks** with description, category, likelihood, and impact
- **Auto-calculates risk score** (Likelihood × Impact) and assigns a rating
- **Filter by severity** — Critical, High, Medium, Low
- **Live dashboard** showing total, critical, and high risk counts
- **Persistent storage** — data saves in your browser across sessions
- **Pre-loaded with 5 sample risks** to demonstrate the tool

---

## 📊 Risk Rating Matrix

| Score | Rating | Response Time |
|-------|--------|---------------|
| 20–25 | 🔴 Critical | Immediate action |
| 15–19 | 🟠 High | Within 7 days |
| 7–14  | 🟡 Medium | Within 30 days |
| 1–6   | 🟢 Low | Within 90 days |

---

## 🎯 GRC Concepts Demonstrated

- **Risk identification and scoring** using a 5×5 likelihood-impact matrix
- **ISO 27001** risk assessment methodology (Clause 6.1.2)
- **NIST Cybersecurity Framework** risk management principles
- **Risk categorization** across 8 security domains

## 🛠️ Tech Stack

- Pure HTML, CSS, JavaScript — zero dependencies
- Browser localStorage for data persistence
- Fully offline — no data leaves your device
